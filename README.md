# Autoresume: AI‑Powered Resume Tailoring

Automated Python tool that tailors LaTeX resumes to job descriptions using Gemini API.

## Features
- Extracts Job Description signals (backend/ML/teamwork/domain) and semantically tags resume bullets
- Weighted scoring algorithm prioritizes high‑signal content
- Generates tailored LaTeX + compiles to PDF via `pdflatex`
- Robust rate limiting, retry logic, JSON parsing
